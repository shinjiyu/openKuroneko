# 需要验证并完成某个里程碑，确认交付物已存在且符合要求

> category: file | id: s-396221 | 2026-03-06T09:15:23.755Z

场景：需要验证并完成某个里程碑，确认交付物已存在且符合要求
步骤：
1. 读取 knowledge.md 检查历史完成记录，避免重复执行
2. 读取实际交付物文件，验证内容完整性
3. 读取 milestones.md 确认当前状态
4. 使用 edit_file 将里程碑状态从 [Active] 同步为 [Completed]
5. 再次读取 milestones.md 二次验证更新成功
验证：milestones.md 显示目标里程碑为 [Completed]，交付物文件存在且内容完整
