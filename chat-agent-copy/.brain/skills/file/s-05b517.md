# 需要验证扫描类里程碑是否已完成（目录结构、文件存在性检查等）

> category: file | id: s-05b517 | 2026-03-06T09:15:23.745Z

场景：需要验证扫描类里程碑是否已完成（目录结构、文件存在性检查等）
步骤：
1. 读取 knowledge.md 获取历史记录，提取已记录的目录结构、文件状态和里程碑完成情况
2. 执行实际扫描命令（如 find . -maxdepth 2）验证当前状态与历史记录一致
3. 读取 milestones.md 确认目标里程碑当前状态
4. 编辑 milestones.md 将目标里程碑从 [Active] 改为 [Completed]
5. 再次读取 milestones.md 验证状态更新成功
验证：通过 knowledge.md 历史记录确认文件已存在，通过 find 命令确认目录结构，通过两次读取 milestones.md 确认状态成功从 [Active] 变为 [Completed]
