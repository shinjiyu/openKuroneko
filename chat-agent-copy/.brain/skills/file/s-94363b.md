# 需要验证某个里程碑的交付物已完成，并更新里程碑状态为 Completed

> category: file | id: s-94363b | 2026-03-06T09:15:23.760Z

场景：需要验证某个里程碑的交付物已完成，并更新里程碑状态为 Completed

步骤：
1. 读取 .brain/milestones.md，确认目标里程碑的当前状态和描述
2. 读取/验证交付物（如文件、文档等），确认存在且内容有效
3. 使用 edit_file 精确替换里程碑状态：将 [Active] 改为 [Completed]
4. 再次读取 milestones.md，确认状态更新成功

验证：里程碑状态已从 Active 变为 Completed，且交付物确实存在并符合预期
