# 需要验证某个交付物文件已创建且内容有效，然后更新里程碑状态

> category: file | id: s-9b4367 | 2026-03-06T09:15:23.762Z

场景：需要验证某个交付物文件已创建且内容有效，然后更新里程碑状态

步骤：
1. 读取里程碑文件，确认当前活跃里程碑的目标描述
2. 读取待验证的交付物文件，检查内容是否完整有效
3. 使用 edit_file 工具，通过 old_string 精确匹配原状态，new_string 设置新状态
4. 再次读取里程碑文件，验证状态更新成功

验证：最终读取的里程碑文件中，目标里程碑状态已从 [Active] 变为 [Completed]
