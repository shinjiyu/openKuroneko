# Playwright 启动浏览器等待人类完成登录操作

> category: browser | id: s-8108c9 | 2026-03-06T09:15:23.765Z

场景：Playwright 启动浏览器等待人类完成登录操作
步骤：
1. 使用 chromium.launch({ headless: false }) 显示浏览器窗口
2. 导航到登录页面
3. 使用轮询方式检查登录成功标志（如特定元素出现）
4. 检测到登录成功后，使用 context.storageState() 保存登录状态到 JSON 文件
5. 使用 await browser.close() 而非 browser.disconnect() 确保脚本正确退出
验证：登录状态文件存在且包含 cookies/localStorage 信息
