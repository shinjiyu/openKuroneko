# 极简项目（1 目录 + 2-3 个核心文件）的关键文件识别与读取任务

> category: file | id: s-267906 | 2026-03-06T09:15:23.748Z

场景：极简项目（1 目录 + 2-3 个核心文件）的关键文件识别与读取任务
步骤：
  1. 读取 knowledge.md 检查历史记录，确认已识别的核心文件列表
  2. 读取 milestones.md 确认当前里程碑状态
  3. 逐个读取核心文件（如 overview.md、project-structure.md、核心业务文件）
  4. 理解项目逻辑：识别项目类型（极简项目）、技术栈（如单文件 Web 应用）、功能模块
  5. 更新 milestones.md 状态（Active → Completed）
  6. 二次验证更新成功
验证：milestones.md 中对应里程碑状态为 [Completed]，且已读取所有核心文件
