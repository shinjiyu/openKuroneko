# 里程碑完成时需要验证交付物并同步状态

> category: file | id: s-7529bb | 2026-03-06T09:15:23.759Z

场景：里程碑完成时需要验证交付物并同步状态
步骤：
1. 读取 milestones.md 确认目标里程碑当前状态
2. 检查交付物文件是否存在（read_file 或 list_directory）
3. 读取交付物文件验证内容完整性（关键字段、结构化内容）
4. 使用 edit_file 更新里程碑状态（Active → Completed）
5. 再次读取 milestones.md 确认更新成功
验证：目标里程碑状态变为 [Completed]，且所有前置里程碑均为 [Completed]
