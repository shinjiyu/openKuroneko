# 需要在工作目录中查找特定主题的数据文件（如查找某个用户、项目或关键词的相关文件）

> category: file | id: s-3787c6 | 2026-03-06T09:15:23.764Z

场景：需要在工作目录中查找特定主题的数据文件（如查找某个用户、项目或关键词的相关文件）
步骤：
1. 使用 find 命令按文件名模式搜索：`find . -type f -name "*关键词*"`
2. 使用 grep 命令在文本文件中搜索内容：`grep -r "关键词" . --include="*.md" --include="*.json" --include="*.txt"`
3. 结合两种方法交叉验证，避免遗漏
4. 如确认不存在，明确记录结果并更新里程碑状态
验证：grep 和 find 命令输出为空，且已在所有合理文件类型中搜索
