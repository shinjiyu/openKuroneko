场景：里程碑完成时的文档验证与状态同步
步骤：
  1. 读取 milestones.md 确认当前任务定义
  2. 读取目标交付文件，验证内容完整性（检查关键章节/要素是否齐全）
  3. 使用 edit_file 更新 milestones.md，将对应里程碑标记为 [Completed]
  4. 再次读取 milestones.md 确认状态更新成功
验证：
  - 交付文件存在且内容符合要求
  - milestones.md 中对应条目状态已变为 [Completed]
  - 无编辑冲突或状态不一致
