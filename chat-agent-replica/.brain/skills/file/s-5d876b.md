场景：执行已完成的里程碑任务（如 M1 目录扫描），需要验证历史记录并同步状态

步骤：
1. 读取 .brain/knowledge.md 检查历史事实，确认任务是否已在之前周期完成
2. 执行最小化验证命令（如 find）确认实际状态未变化
3. 使用 edit_file 将里程碑状态从 [Active] 更新为 [Completed]
4. 使用 read_file 二次验证 milestones.md 确认状态更新成功

验证：read_file(.brain/milestones.md) 显示目标里程碑状态为 [Completed]，且未产生重复的历史记录写入
