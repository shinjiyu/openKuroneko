# 需要从有登录墙限制的社交平台（如微博）获取用户公开数据

> category: browser | id: s-990911 | 2026-03-06T09:15:23.765Z

场景：需要从有登录墙限制的社交平台（如微博）获取用户公开数据
步骤：
  1. 先读取已有的调查报告，了解历史尝试和结论
  2. 使用搜索引擎搜索 "用户名 平台名" 和 "site:platform.com 用户名"
  3. 直接访问平台用户主页 URL（如 weibo.com/u/用户名）
  4. 使用 Playwright 等浏览器自动化工具尝试绕过简单限制
  5. 记录所有尝试结果，特别标注返回的登录/验证页面
  6. 更新 knowledge.md 记录调查过程和关键发现
  7. 如所有方法均被登录墙阻挡，记录为平台限制并标记需要人工协助
验证：
  - 如平台返回 "Sina Visitor System"、"需登录"、"Sign in" 等明确登录要求
  - 且搜索引擎未返回用户主页的公开缓存或镜像
  - 则确认为平台强制登录限制，需要 BLOCK 等待人类提供登录或用户 ID
