# 多源信息整合与用户画像综合分析

> category: data | id: s-406658 | 2026-03-06T10:06:39.445Z

场景：需要基于多个分散的数据文件（信息汇总、用户画像、平台详情等）形成完整的用户综合分析报告

步骤：
1. 读取所有相关数据文件（信息汇总、概览、用户画像、平台详情、分析结论等）
2. 识别关键分析维度（活跃度、生活状态、职业发展、社交网络、心理画像）
3. 对每个维度进行跨源交叉验证，标注信息可靠性（⭐分级）
4. 采用结构化输出：执行摘要 → 分维度详细分析 → 综合结论 → 建议与后续行动
5. 明确标注信息完整度、数据缺口、分析局限性
6. 对不确定信息使用"推测"、"可能"等措辞，避免过度解读

验证：
- 报告覆盖所有要求的分析维度
- 每项结论都有明确的数据来源和可靠性标注
- 结构完整，包含摘要、详细分析、结论、建议
- 明确标注了信息缺口和后续需求
