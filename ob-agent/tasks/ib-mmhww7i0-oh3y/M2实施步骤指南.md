## 📋 Token 获取后的验证步骤

### 1. 保存 Token

```bash
# 保存到环境变量
export XIAOMI_CAMERA_TOKEN="获取到的32位Token"
export XIAOMI_CAMERA_IP="192.168.31.25"

# 或保存到文件
echo "你的Token" > .xiaomi_camera_token
```

### 2. 验证 Token 有效性

```bash
# 测试设备连接
miiocli device --ip 192.168.31.25 --token YOUR_TOKEN info

# 如果成功，会显示设备型号、固件版本等信息
```

### 3. 探索设备功能

```bash
# 列出所有属性
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN properties

# 列出所有操作
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN actions

# 查找 PTZ 相关功能
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN properties | grep -i ptz
```

## 🔄 备选方案（如果以上方案无法使用）

### 方案 D: iPhone 备份提取（需要 iPhone）

1. iPhone 连接电脑，使用 iTunes/Finder 进行**未加密备份**
2. 下载 iBackup Viewer（https://www.imactools.com/iphonebackupviewer/）
3. 打开备份，找到 `xiaomi.mihome/Documents/XXX_mihome.sqlite`
4. 查看 `ZDEVICE` 表，找到 `ZTOKEN` 字段

### 方案 E: 设备重置绑定法（最简单但需要重置设备）

1. 长按摄像头重置按钮 5 秒
2. 等待设备重启完成（约 30 秒）
3. 运行以下命令抓取 token：

```bash
# 在设备重置后立即运行
miiocli discover --handshake 1
# 如果设备固件较旧（3.3.9_003077 以下），可能会显示 token
```

**注意**: 此方法在较新固件上已被封堵

## ⚠️ 常见问题

### Q1: 工具提示 "Unable to log in"
**A**: 可能原因：
- 账号密码错误
- 小米账号被风控
- 需要先在米家 App 中登录一次

### Q2: 提示需要短信验证码
**A**: 正常情况，输入收到的验证码即可

### Q3: 找不到摄像头设备
**A**: 
- 确保摄像头已添加到米家 App
- 尝试选择不同的服务器地区
- 检查设备是否在线

### Q4: Token 获取后无法连接
**A**: 
- 检查 IP 地址是否正确
- 确认 Token 复制完整（32 位十六进制）
- 尝试重启设备
