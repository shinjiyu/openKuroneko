## 📋 操作步骤

无论选择哪种方式，操作流程相同：

1. **选择登录方式**: 输入 `p`（密码登录）或 `q`（二维码登录）
2. **输入账号**: 小米账号（手机号或邮箱）
3. **输入密码**: 小米账号密码
4. **选择服务器**: 输入 `cn`（中国区）
5. **验证码**: 可能需要短信验证码
6. **获取 Token**: 从输出中找到 IP 为 192.168.31.25 的设备，复制其 Token

## 🚫 无法自动完成的原因

Token 获取工具需要：
1. **交互式输入**: 账号、密码、验证码等
2. **用户授权**: 可能需要手机短信验证
3. **安全考虑**: 小米账号密码不应硬编码

这些因素导致无法通过自动化脚本完成，需要用户手动操作。
