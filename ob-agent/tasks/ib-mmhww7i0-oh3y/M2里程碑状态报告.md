## 📋 Token 获取后的验证流程

用户获取 Token 后，执行以下验证：

### 1. 保存 Token
```bash
export XIAOMI_CAMERA_TOKEN="你的32位Token"
export XIAOMI_CAMERA_IP="192.168.31.25"
```

### 2. 验证连接
```bash
miiocli device --ip 192.168.31.25 --token $XIAOMI_CAMERA_TOKEN info
```

### 3. 探索功能
```bash
# 列出所有属性
miiocli genericmiot --ip 192.168.31.25 --token $XIAOMI_CAMERA_TOKEN properties

# 列出所有操作
miiocli genericmiot --ip 192.168.31.25 --token $XIAOMI_CAMERA_TOKEN actions

# 查找 PTZ 相关功能
miiocli genericmiot --ip 192.168.31.25 --token $XIAOMI_CAMERA_TOKEN properties | grep -i ptz
```

## 🔍 技术发现

### miIO 协议要求
- Token 是 32 位十六进制字符串
- 用于加密 miIO 协议通信
- 每个设备唯一，通常不变
- 重置设备或更换 WiFi 可能会改变

### 小米安全机制
- 新固件已封堵无需账号的 token 获取方法
- 云端 API 需要账号认证
- 可能触发短信验证码
- 某些账号可能被风控
