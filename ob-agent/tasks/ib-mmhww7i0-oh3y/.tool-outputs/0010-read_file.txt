## 关键技术发现

### ⚠️ 重要限制
1. **小米摄像头不支持 ONVIF 协议**（官方封闭生态策略）
2. **部分型号支持 RTSP**（需要特定方法开启）
3. **PTZ 控制需要通过 miIO/MIoT 协议实现**

### ✅ 可行方案
1. **python-miio** - 本地控制（需要设备 token）
2. **小米官方 HA 集成** - 云端 API
3. **hass-xiaomi-miot** - 社区增强版
4. **micam** - RTSP 视频流桥接

## 快速开始命令

### 安装 python-miio
```bash
pip install python-miio
```

### 发现设备
```bash
miiocli discover
```

### 获取设备信息（需要 token）
```bash
miiocli genericmiot --ip 192.168.1.100 --token YOUR_TOKEN info
```

### 列出设备属性
```bash
miiocli genericmiot --ip 192.168.1.100 --token YOUR_TOKEN properties
```

### 列出设备动作
```bash
miiocli genericmiot --ip 192.168.1.100 --token YOUR_TOKEN actions
```
