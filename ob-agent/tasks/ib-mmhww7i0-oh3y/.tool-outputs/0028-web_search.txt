It supports two ways of authentication: username (e-mail/Xiaomi Cloud account ID) &amp; password QR code After logging in you have to select a Xiaomi's server region ( cn - China, de - Germany etc.). Leave it empty to check all available In return all of your devices connected to account will be listed, together with their name and IP address. Windows Download and run token_extractor.exe .
