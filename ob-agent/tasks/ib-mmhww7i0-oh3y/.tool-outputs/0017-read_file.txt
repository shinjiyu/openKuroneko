## 六、关键技术发现

### 6.1 协议特性
1. **miIO 协议**:
   - 使用 UDP 54321 端口
   - 需要设备 token 进行认证
   - Token 是 32 字节的十六进制字符串

2. **RTSP 视频流**:
   - 默认关闭，需要手动开启
   - 可通过米家 App 或 miIO 命令开启
   - 开启后可通过标准 RTSP 客户端获取视频流

3. **PTZ 控制**:
   - 需要通过 miIO 协议调用
   - python-miio 库可能不直接支持 PTZ
   - 可能需要使用 `genericmiot` 接口

### 6.2 安全机制
- 设备不响应未授权的 miIO 命令
- Token 存储在小米云端，需要账号登录获取
- RTSP 流可能需要认证

miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN actions | grep -i ptz
