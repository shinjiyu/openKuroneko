## 二、技术验证

### 2.1 miIO 协议验证

**测试代码**: `test_miio_connection.py`

```python
# 发送发现包
message = bytes.fromhex("213100200000000000000000ffffffffffffffffffffffffffffffff")
sock.sendto(message, (target_ip, 54321))

# 收到响应
response, addr = sock.recvfrom(1024)
# 响应: 21310020000000000ed8c6bc006cec68ffffffffffffffffffffffffffffffff
```

**结论**: miIO 协议可达，可以与设备通信。

### 2.2 端口扫描验证

```bash
# TCP 端口扫描
nmap -p 80,443,554,8080,8443,8554,9000,9999 192.168.31.25
# 结果: 所有端口关闭

# UDP 端口测试
nc -vzu 192.168.31.25 54321
# 结果: Connection succeeded!
```

**结论**: 仅 UDP 54321 端口开放，符合 miIO 协议特征。

## 三、受阻原因分析

### 3.1 Token 获取

**问题**: miIO 协议需要 token 进行认证  
**影响**: 无法调用任何设备功能  
**解决方案**: 使用 Xiaomi Cloud Tokens Extractor 获取

### 3.2 RTSP 功能

**问题**: RTSP 端口默认关闭  
**影响**: 无法获取视频流  
**解决方案**: 在米家 App 中开启 RTSP 功能

### 3.3 PTZ 控制

**问题**: 需要 token 才能测试  
**影响**: 无法验证 PTZ 控制功能  
**解决方案**: 获取 token 后使用 miiocli 测试

## 四、下一步行动计划

### 4.1 立即行动（优先级：高）

#### 步骤 1: 获取设备 Token

**方法 1: 使用 Docker 工具（推荐）**
```bash
docker run --rm -it pymoroni/xiaomi-cloud-tokens-extractor
```

**方法 2: 使用 Python 脚本**
```bash
pip install micloud
python get_token.py --username YOUR_USERNAME --password YOUR_PASSWORD
```

#### 步骤 2: 测试 miIO 功能

```bash
# 测试设备信息
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN info

# 列出所有属性
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN properties

# 列出所有动作
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN actions
```

### 4.2 短期行动（优先级：中）

#### 步骤 3: 开启 RTSP 功能

**方法 1: 米家 App**
- 打开米家 App → 选择摄像头 → 设置 → 存储管理 → RTSP 流 → 开启

**方法 2: miIO 命令**
```bash
# 查找 RTSP 相关属性
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN properties | grep -i rtsp

# 尝试开启 RTSP
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN set rtsp_enabled true
```

#### 步骤 4: 测试视频流

```bash
# 使用 ffplay 测试
ffplay rtsp://192.168.31.25:554/live/ch00_0

# 使用 VLC 测试
vlc rtsp://192.168.31.25:554/live/ch00_0

# 使用 ffmpeg 录制
ffmpeg -i rtsp://192.168.31.25:554/live/ch00_0 -c copy test.mp4
```

### 4.3 中期行动（优先级：低）

#### 步骤 5: 测试 PTZ 控制

```bash
# 查找 PTZ 相关属性和动作
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN properties | grep -i ptz
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN actions | grep -i ptz

# 尝试 PTZ 控制（假设存在 ptz_move 动作）
miiocli genericmiot --ip 192.168.31.25 --token YOUR_TOKEN call ptz_move '{"direction": "left", "speed": 5}'
```
