## 任务目标
研究如何用代码控制局域网内的小米摄像头，并尝试连接当前局域网内的摄像头设备。

## 具体步骤
1. **搜索解决方案**
   - 搜索小米摄像头的 API 文档和开发者资源
   - 查找开源项目和控制方案（如 miio 协议、homeassistant 集成等）
   - 了解认证方式和通信协议

2. **分析技术方案**
   - 确定可行的控制方法（官方 API / 第三方库 / 协议逆向）
   - 整理所需工具和依赖
   - 识别潜在的技术难点

3. **尝试连接**
   - 扫描局域网内的小米摄像头设备
   - 获取摄像头 IP 和设备信息
   - 尝试建立连接并测试基本控制功能（如获取视频流、云台控制等）

4. **输出结果**
   - 提供可行的技术方案
   - 记录连接过程中的问题和解决方案
   - 如成功连接，提供控制代码示例

## 约束条件
- 仅用于合法用途和个人设备控制
- 注意隐私和安全，不进行未授权的访问
- 优先使用官方或开源的成熟方案
