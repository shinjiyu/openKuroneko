# IoT 设备 miIO 协议连接验证流程

> category: general | id: iot-miio-affa | 2026-03-08T16:03:36.130Z

场景：需要验证小米或支持 miIO 协议的 IoT 设备是否可通信，并测试基础连接能力

步骤：
  1. 网络连通性测试：使用 ping 和 arp 确认设备在线并获取 MAC 地址
  2. UDP 端口验证：使用 nc -vzu <IP> 54321 测试 miIO 标准端口是否开放
  3. 发送 miIO 发现包：构造 32 字节发现包（21310020...）发送到 UDP 54321
  4. 解析响应：检查是否收到 32 字节响应，提取设备 ID（字节 8-12）
  5. TCP 端口扫描：使用 nmap 扫描常用端口（80/443/554/8080 等）确认设备开放的服务
  6. 判断协议支持：如果收到有效响应，确认设备支持 miIO 协议
  7. 识别阻塞项：如果需要 token 认证，提示使用 Xiaomi Cloud Tokens Extractor 获取

验证：
  - 收到 32 字节 miIO 响应（格式：21310020 + 设备信息）
  - 能够提取设备 ID（4 字节十六进制）
  - 确认 UDP 54321 端口开放
  - 明确后续需要的认证信息（token/登录等）
