安装 Edge TTS uv tool install edge-tts # 安装 ffmpeg（如果未安装） sudo apt install ffmpeg 2. 配置飞书应用 创建配置文件 ~/.openclaw/workspace/config/feishu_voice.json ： { "app_id" : " your_app_id " , "app_secret" : " your_app_secret " } 获取飞书应用凭证： 访问 飞书开放平台 创建企业自建应用 获取 App ID 和 App Secret 开通权限： im:file:read 、 im:file:write 、 im:message:send_as_bot 3. 发送语音消息 # 使用默认语音（晓晓） bash send_voice.sh " 你好，这是一条语音消息 " ou_xxx # 使用其他语音 bash send_voice.sh " 早上好 " ou_xxx zh-CN-YunxiNeural 可用语音 语音 ID 名称 性别 风格 zh-CN-XiaoxiaoNeural 晓晓 女 温暖 ✅ 默认 zh-CN-XiaoyiNeural 晓伊 女 活泼 zh-CN-YunxiNeural 云希 男 阳光 zh-CN-YunyangNeural 云扬 男 专业 zh-CN-liaoning-XiaobeiNeural 辽宁小贝 女 东北话 zh-CN-shaanxi-XiaoniNeural 陕西小尼 女 陕北话 Python API from send_voice import send_voice_message # 发送给私聊用户 message_id = send_voice_message ( text = "你好，我是清道夫" , open_id = "ou_xxx" ) # 发送给群聊 message_id = send_voice_message ( text = "大家好" , chat_id = "oc_xxx" , voice = "zh-CN-YunxiNeural" ) 命令行参数 python3 send_voice.py --help # 参数说明 --text 要转换的文本（必需） --open-id 私聊用户的 open_id --chat-id 群聊的 chat_id --voice TTS 语音名称（默认：晓晓） -v, --verbose 输出详细信息 技术流程 文本 → Edge TTS → MP3 → ffmpeg → opus → 飞书上传 → 发送语音消息 生成语音 ：Edge TTS 将文本转为 MP3 转换格式 ：ffmpeg 将 MP3 转为 opus（飞书推荐格式） 上传文件 ：飞书 API 上传获取 file_key 发送消息 ：飞书 API 发送 audio 类型消息 注意事项 私聊使用 open_id ，群聊使用 chat_id 飞书语音消息最大时长：30 分钟 文本长度建议：不超过 500 字（避免超时） 需要开通飞书应用的消息发送权限 文件结构 feishu-voice/ ├── README.md #
