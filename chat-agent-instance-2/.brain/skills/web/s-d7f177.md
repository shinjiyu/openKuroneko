# 需要通过搜索引擎探索某用户名在多个社交平台的信息（OSINT 调查）

> category: web | id: s-d7f177 | 2026-03-06T09:15:23.764Z

场景：需要通过搜索引擎探索某用户名在多个社交平台的信息（OSINT 调查）
步骤：
  1. 使用 web_search 搜索 "<username> 微博" 和 "<username> social media profile" 获取初步结果
  2. 使用 web_search 搜索 "<username> Twitter OR X platform profile" 和 "<username> LinkedIn Facebook profile" 扩展搜索
  3. 使用 web_search fetch 操作尝试访问发现的社交媒体链接
  4. 记录所有发现的相似账号信息（平台、用户名、粉丝数、简介等）
  5. 评估数据获取可行性（平台访问限制、账号存在性、信息确认度）
验证：milestones.md 状态更新为 [Completed]，knowledge.md 包含完整的信息源发现记录
