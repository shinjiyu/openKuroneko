# 完成一个里程碑后需要验证交付物并更新里程碑状态

> category: file | id: s-6262a9 | 2026-03-06T09:15:23.763Z

场景：完成一个里程碑后需要验证交付物并更新里程碑状态
步骤：
1. 读取交付物文件，验证文件存在性和内容有效性（检查关键字段/章节）
2. 读取 .brain/milestones.md，确认当前里程碑状态
3. 使用 edit_file 工具，将目标里程碑的状态标记从 [Active] 替换为 [Completed]
4. 再次读取 milestones.md 验证更新成功
验证：交付物文件存在且内容有效 + milestones.md 中对应里程碑显示 [Completed]
