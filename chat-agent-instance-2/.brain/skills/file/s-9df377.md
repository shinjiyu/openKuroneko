# 需要验证某个里程碑是否已完成，避免重复执行扫描/生成任务

> category: file | id: s-9df377 | 2026-03-06T09:15:23.743Z

场景：需要验证某个里程碑是否已完成，避免重复执行扫描/生成任务

步骤：
1. 读取 .brain/knowledge.md，搜索"里程碑完成"或"[M1-M4]"相关记录
2. 检查记录时间戳和完成内容（如"M4 里程碑完成：overview.md 已更新为完整的项目概览文档"）
3. 读取 .brain/milestones.md，验证状态标记是否为 [Completed]
4. 如发现状态不一致（knowledge.md 记录完成但 milestones.md 仍为 Active），执行 edit_file 同步状态
5. 如已完成，跳过重复任务执行；如未完成，按计划执行

验证：通过 knowledge.md 历史记录确认任务已在之前周期完成，milestones.md 状态与 knowledge.md 一致
