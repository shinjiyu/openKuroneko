# 里程碑验证任务（检查目录结构、文件存在性、避免重复工作）

> category: file | id: s-f4496e | 2026-03-06T09:15:23.744Z

场景：里程碑验证任务（检查目录结构、文件存在性、避免重复工作）

步骤：
1. 读取 .brain/knowledge.md 获取历史项目事实记录
2. 在历史记录中搜索目标验证项（如"目录结构"、"文件存在"、"里程碑完成"）
3. 如果历史记录已包含完整信息，直接确认完成
4. 仅当历史记录缺失或不完整时，才执行实际的文件系统扫描
5. 读取 .brain/milestones.md 确认当前里程碑状态
6. 如果实际已完成但状态未同步，执行 edit_file 将 [Active] 改为 [Completed]
7. 读取验证修改成功

验证：
- knowledge.md 历史记录包含目标验证项的完整描述
- milestones.md 状态已同步为 [Completed]
- 整个过程未调用冗余的扫描工具（如 list_directory、read_file 读取已记录的文件）
