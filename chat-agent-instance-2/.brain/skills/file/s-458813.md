# 需要建立项目目录结构文档

> category: file | id: s-458813 | 2026-03-06T09:15:23.741Z

场景：需要建立项目目录结构文档
步骤：
  1. 使用 `find . -type f -o -type d | head -100` 获取完整文件列表
  2. 使用 `find . -maxdepth 3 -type d | sort` 获取目录层级
  3. 使用 `find . -maxdepth 3 -type f | sort` 获取文件列表
  4. 读取现有的 project-structure.md 和 knowledge.md 确认分析状态
  5. 更新文档，包含：目录树、文件统计表、层级关系图
验证：
  - 文档包含完整的目录树（tree 格式）
  - 文档包含文件统计表（类型/数量/文件列表）
  - 文档包含层级关系图（第 0/1/2 层结构）
