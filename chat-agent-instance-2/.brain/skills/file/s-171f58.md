# 里程碑状态不一致（knowledge.md 记录任务完成，但 milestones.md 状态仍为 Active/Pen

> category: file | id: s-171f58 | 2026-03-06T09:15:23.744Z

场景：里程碑状态不一致（knowledge.md 记录任务完成，但 milestones.md 状态仍为 Active/Pending）
步骤：
1. 读取 knowledge.md 历史记录，确认任务完成时间点和完成证据
2. 读取 milestones.md 当前状态，识别状态不一致的里程碑项
3. 执行状态同步（edit_file: [Active]/[Pending] → [Completed]）
4. 验证更新成功（重新读取 milestones.md 确认状态已变更）
验证：milestones.md 状态与 knowledge.md 历史记录一致，且无重复扫描操作
