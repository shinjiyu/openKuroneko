# 需要识别并读取项目中的关键文件，同时管理里程碑状态

> category: file | id: s-5006e2 | 2026-03-06T09:15:23.758Z

场景：需要识别并读取项目中的关键文件，同时管理里程碑状态

步骤：
1. 读取 .brain/milestones.md 确认当前里程碑定义和目标
2. 根据里程碑目标，识别需要读取的文件列表
3. 逐个读取关键文件，分析其类型和作用
4. 使用 edit_file 更新里程碑状态（将 [Active] 改为 [Completed]）
5. 读取验证里程碑文件，确认状态更新成功

验证：里程碑文件中目标项状态已从 [Active] 变为 [Completed]，且所有目标文件已成功读取
