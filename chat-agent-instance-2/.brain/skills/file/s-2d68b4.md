# 需要快速扫描目录结构并建立项目骨架

> category: file | id: s-2d68b4 | 2026-03-06T09:15:23.741Z

场景：需要快速扫描目录结构并建立项目骨架
步骤：
  1. 使用 `find . -type f -o -type d` 递归列出所有文件和目录
  2. 使用 `find . -maxdepth 3 -type d | sort` 分离目录结构
  3. 使用 `find . -maxdepth 3 -type f | sort` 分离文件列表
  4. 读取已有的 project-structure.md 确认是否需要更新
  5. 优化文档格式（如文件统计表）提升可读性
验证：文档包含完整目录树、文件统计表、层级关系图
