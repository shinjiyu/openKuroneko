# 需要扫描并分析项目目录结构，建立层级关系

> category: file | id: s-836309 | 2026-03-06T09:15:23.739Z

场景：需要扫描并分析项目目录结构，建立层级关系
步骤：
  1. 使用 `find . -type f -o -type d | head -100` 递归列出所有文件和目录
  2. 读取已有文档（如 project-structure.md, knowledge.md）了解历史记录
  3. 生成结构报告，包含：目录树、文件统计、**层级关系图**（标注第N层）、核心文件说明
验证：结构报告包含完整的层级关系图，且所有已知文件都被列出
