# 目标用户信息不可获取时，需要输出任务终止说明报告

> category: data | id: s-39aa8d | 2026-03-06T09:15:23.764Z

场景：目标用户信息不可获取时，需要输出任务终止说明报告
步骤：
1. 读取 knowledge.md 历史记录，提取前序里程碑收集的所有碎片化信息
2. 读取信息概览文档（如 steph808-info-overview.md），获取结构化的信息汇总
3. 生成分析结论报告，包含以下关键章节：执行摘要、数据源调查结果、可行性评估、任务终止条件、恢复任务的条件、关键事实清单
4. 更新里程碑状态（[Active] → [Completed]）并进行二次验证
验证：报告文件存在且包含完整的任务终止说明，里程碑状态已更新为 Completed
