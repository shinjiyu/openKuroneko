# 需要调查某个用户名在社交媒体平台的公开信息，但目标平台有登录限制

> category: browser | id: s-582b95 | 2026-03-06T09:15:23.765Z

场景：需要调查某个用户名在社交媒体平台的公开信息，但目标平台有登录限制

步骤：
1. 先读取已有调查文档，避免重复工作
2. 搜索第三方存档（Web Archive、Google Cache、Archive.today）
3. 搜索跨平台公开索引（约会网站、论坛、其他社交媒体）
4. 搜索转发引用（使用 site: 搜索其他平台的提及）
5. 使用 playwright 访问需要 JavaScript 的页面
6. 记录每个平台的访问状态、可获取信息、身份特征
7. 对比不同平台信息的一致性（地理位置、年龄、姓名）
8. 评估跨平台账号与目标平台的关联性

验证：
- 成功获取至少 3 个平台的公开信息
- 记录所有尝试的存档服务结果
- 明确标注信息冲突和关联性评估
- 更新 knowledge.md 记录调查过程和结论
